# Creative Loading Animation

A simple and elegant loading animation built with HTML and CSS.

## Features

- Smooth pulsing animation effect
- Four animated dots with staggered delays
- Responsive design
- Clean and minimal code

## How It Works

The animation consists of four white circular dots that pulse in sequence, creating a smooth loading effect. Each dot has a different animation delay, creating a wave-like pattern.

### Animation Details

- **Duration**: 0.8 seconds per cycle
- **Effect**: Dots scale from 30px to 20px and back
- **Stagger**: Each dot delays by 0.1-0.2 seconds for a flowing effect

## Files

- `index.html` - Main HTML structure
- `style.css` - Animation and styling
- `README.md` - This file

## Usage

Simply open `index.html` in your browser to see the loading animation in action.

## Customization

You can customize the animation by modifying:

- **Colors**: Change the `background` in `body` or `background` in `.loader .loading`
- **Size**: Adjust `width` and `height` values in the `.loader .loading` class
- **Speed**: Modify the `0.8s` value in the `animation` property
- **Spacing**: Change `margin` or `padding` values
