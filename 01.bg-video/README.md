# Background Video Hero Section

A modern, responsive hero section featuring a full-screen background video with overlay text and call-to-action button.

## Features

- Full-screen hero section with video background
- Responsive design that adapts to different aspect ratios
- Smooth hover animations on text and buttons
- Dark overlay for improved text readability
- Mobile-friendly layout

## Files

- `index.html` - Main HTML structure
- `style.css` - Styling and animations
- `assets/video.mp4` - Background video file

## Setup

1. Replace the video file at `assets/video.mp4` with your own video
2. Update the heading text and button link as needed
3. Open `index.html` in your browser

## Customization

- **Title**: Change the text inside `<h1>` tag in `index.html`
- **Button Text**: Modify the anchor tag text
- **Button Link**: Update the `href` attribute in the anchor tag
- **Colors**: Adjust the gradient overlay and text colors in `style.css`
- **Video**: Place your video file in the `assets/` folder

## Browser Support

Works on all modern browsers that support HTML5 video and CSS3 animations.
